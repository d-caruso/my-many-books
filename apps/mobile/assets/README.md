# Assets

This folder contains app assets like images, icons, and splash screens.

## Required Files

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen image
- `favicon.png` - Web favicon

## Guidelines

- Use PNG format for icons
- Ensure proper transparency for icons
- Follow platform-specific guidelines for icon design
- Optimize images for file size

## Generating Icons

You can use Expo's icon generator or create custom icons following the platform guidelines:

- iOS: https://developer.apple.com/design/human-interface-guidelines/app-icons
- Android: https://developer.android.com/guide/practices/ui_guidelines/icon_design_adaptive
